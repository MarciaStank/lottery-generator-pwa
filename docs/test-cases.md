# Test Scenarios

## CT01 - Generate lottery with 6 numbers
Given the user selects 6 numbers  
When the user generates the game  
Then the system should return 6 unique numbers  

## CT02 - Prevent duplicate numbers
Given duplicate numbers are entered  
Then the system should display an error  

## CT03 - Validate number limits
The system should allow only between 6 and 9 numbers
